# Slides: From Chatbots to Coding Agents

A short talk for grad students in biomedical research who already use chatbots
and know a little R. It expands the concepts in the top-level `README.md` into a
presentation: chatbot vs. agent, tokens and the context window, cost, and the
habits for working well with context (Markdown memory, clearing context, skills
vs. one large instruction file).

## Build

Built with [Quarto](https://quarto.org/) + reveal.js. To render the deck:

```bash
quarto render slides.qmd --to revealjs
# produces a self-contained slides.html (open it in any browser)
```

Present with `quarto preview slides.qmd`, or open `slides.html` directly.
Speaker notes are in the deck — press **`s`** in the browser to open the
speaker view. The rendered `slides.html` is git-ignored (regenerate it locally).

## Figures

`figures/` holds the explainer graphics, on a shared palette
(input/context = blue, output = amber, good/lean = green, bloat = coral):

| File | What it shows |
| --- | --- |
| `fig-chatbot-vs-agent.svg` | chatbot prompt window vs. the agent plan/act/observe loop |
| `fig-tokens.svg` | a sentence split into tokens; the ~4-chars rule |
| `fig-context-window.svg` | the context window as a finite, shared token budget |
| `fig-markdown-memory.svg` | a versioned instruction file loaded into every session |
| `fig-context-management.svg` | one bloated session vs. clearing between tasks |
| `fig-skills-vs-monolith.svg` | one large file vs. a small core + skills on demand |
| `chart-cost.svg` | input vs. output token pricing (generated by `charts.R`) |
| `chart-context-rot.svg` | lost-in-the-middle + degradation with length (`charts.R`) |

The two charts are generated with R/ggplot2 (the audience knows R):

```bash
cd figures && Rscript charts.R   # needs ggplot2, patchwork; writes the two chart-*.svg
```

The hand-built `fig-*.svg` files are edited directly. All figures use light
backgrounds and are shown as light "cards" on the dark deck (see `custom.scss`).

Figure numbers (tokens, prices, context-window sizes) are mid-2026 and
illustrative — the shapes and ratios are the durable points.
